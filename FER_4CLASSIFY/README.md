#### FER_4Classify

* 方法类型：

  * 基于深度特征提取：
    * 人脸检测、对齐
    * 提特征、分类
  * 基于AU检测方法
    * 基于FACS（面部各个部位运动），如传统的AU检测
    * 基于自监督深度学习方法，如中科院提出的TCAE方法
